# rutube-download-from-tv
Два скрипта. Первый(parse_rutube_series) парсит страницы сериала вида rutube.ru/metainfo/tv/123456 , второй (download_series_range) делает массовое скачивание видео по ссылкам из файла links.txt. Всё запускаем через консоль.
Ссылки на сериалы надо в коде менять. Я в вас верю. Удачи.
Не забудьте bs4 установить.
